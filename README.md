# learn-vibe-coding

Fondasi project backend menggunakan Bun, ElysiaJS, Drizzle, dan MySQL.

## Jalankan

```bash
bun install
bun dev
```

## Script penting

```bash
bun test
bun run typecheck
bun run db:generate
bun run db:push
```

## Konfigurasi

Salin `.env.example` menjadi `.env`, lalu sesuaikan koneksi MySQL sesuai lingkungan kamu.
